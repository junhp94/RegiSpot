# Register Session

RegiSpot is a personal serverless web application that allows groups of people to register for sports sessions (e.g. badminton) with real-time capacity tracking.

The goal of this project is to practice AWS serverless architecture, frontendâ€“backend integration, and cloud deployment using modern web technologies.

* Frontend
    - React + Vite 
    - Hosted on Amazon S3
    - Amazon CloudFront (HTTPS support)

* Backend (serverless)
    - Amazon API Gateway
    - AWS Lambda
    - DynamoDB

* Infrastructure & DevOps
    - AWS SAM
    - IAM
